# 音声ファイル

## 必要な音声ファイル

1. **正解音**: `correct.mp3` - 正解時の音
2. **不正解音**: `beep4.mp3` - 不正解時の音

## ファイル配置

```
assets/sounds/
├── correct.mp3    # 正解音
└── beep4.mp3      # 不正解音
```

## 使用方法

音声ファイルを上記の場所に配置してください。

## 音声仕様

- **フォーマット**: MP3
- **長さ**: 1-3秒程度
- **音量**: 適度な音量（大きすぎない）
- **内容**:
  - `correct.mp3`: 明るく短い正解音
  - `beep4.mp3`: 短いビープ音

## 実装済み機能

- ✅ 音声効果の再生
- ✅ 音声のオン/オフ切り替え
- ✅ 設定の永続化
- ✅ プロフィール画面での設定変更
